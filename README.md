Real-Time Driver Fatigue Monitoring System
Built a real-time AI-powered driver monitoring system using OpenCV, MediaPipe, and Eye Aspect Ratio (EAR) analysis to detect fatigue with low-latency webcam processing and automated safety alerts.
Developed a scalable web application using Streamlit with Progressive Web App (PWA) capabilities, supporting offline access, cross-platform compatibility, and fast application startup.
Implemented facial landmark detection, blink analysis, and optimized computer vision pipelines to improve detection accuracy while maintaining consistent real-time performance.
Designed a modular and extensible architecture with efficient resource management, reusable components, and browser-based deployment, enabling scalable maintenance and future feature integration.
Tech Stack

Python • OpenCV • MediaPipe • Streamlit • NumPy • Pandas • Computer Vision • Eye Aspect Ratio (EAR) • PWA • Git • GitHub
